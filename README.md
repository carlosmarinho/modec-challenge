# Installation

git clone https://github.com/carlosmarinho/modec-challenge.git

Run the following commands

 - npm run install
 - npm run start

## `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

# Tecnologies used

 - React
 - Redux
 - Redux Thunk
 - Styled Components
 - react-leaflet
